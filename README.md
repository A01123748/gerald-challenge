# Gerald-Challenge

Gerald-Challenge is a react-native example of how to use react-native-reanimated library created with expo for ease of use

## Installation

Install [expo](https://docs.expo.dev/get-started/installation/).

```bash
npx expo start
```

With expo installed
```bash
npm run start
npm run android
npm run ios
npm run web
```
